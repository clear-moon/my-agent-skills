# Example Upload Skill

## 概述

这是一个用于演示上传流程的示例技能，不承载复杂业务逻辑，主要用于验证以下事项：

- 技能目录结构是否正确
- `SKILL.md` 是否能被正常收录
- `README.md` 中的技能索引是否同步更新
- Git 提交与推送流程是否顺畅

## 使用方式

当用户明确表示想验证技能上传流程、需要一个最小技能样例，或希望快速确认技能仓库结构是否正确时，可以使用此技能作为模板。

## 执行建议

1. 检查技能目录是否位于 `skills/<skill-name>/`。
2. 确认目录下至少存在一个 `SKILL.md` 文件。
3. 更新仓库 `README.md` 中的技能列表和安装示例。
4. 提交 Git 变更并推送到远程仓库。

## 输出结果

完成后，应能在技能仓库中看到新增的技能目录，并且可以通过 README 中的路径访问到该技能。
